# 机械方块贴图创建指南

## 概述
本指南将帮助您为机械方块创建贴图，使其在游戏中正确显示。

## 贴图文件结构
在Mindustry模组中，方块贴图通常按照以下结构组织：
```
assets/sprites/blocks/[方块名称]/[方块名称].png
```

例如：
- 水车: `assets/sprites/blocks/water_wheel/water_wheel.png`
- 应力源: `assets/sprites/blocks/stress_source/stress_source.png`
- 机械轴: `assets/sprites/blocks/mechanical_shaft/mechanical_shaft.png`

## 贴图尺寸要求
- 1x1方块（如机械轴）: 32x32像素
- 2x2方块（如水车、应力源）: 64x64像素

## 贴图设计建议

### 水车 (water_wheel)
- 设计成水车的外观，包括叶片和中心轴
- 可以考虑添加水花效果作为额外贴图（可选）
- 尺寸: 64x64像素

### 应力源 (stress_source)
- 设计成看起来像是能量源或动力源
- 可以添加一些能量指示灯或旋转元素
- 尺寸: 64x64像素

### 机械轴 (mechanical_shaft)
- 设计成简单的轴或杆状物
- 可以添加一些连接细节，使其看起来能与其它方块连接
- 尺寸: 32x32像素

## 贴图创建步骤

1. **创建基础图像**
   - 使用图像编辑软件（如Photoshop、GIMP或Aseprite）
   - 创建正确尺寸的透明背景图像

2. **设计方块外观**
   - 绘制方块的基础形状
   - 添加细节和纹理
   - 确保风格与Mindustry原版方块一致

3. **添加阴影和高光**
   - 在方块底部添加轻微阴影
   - 在顶部添加高光效果，增强立体感

4. **保存为PNG**
   - 确保保存为PNG格式，保留透明背景
   - 文件名必须与方块名称完全一致（小写，下划线分隔）

## 高级贴图技巧

### 动画效果
如果需要方块有动画效果（如旋转），可以创建额外的贴图：
- 旋转部分: `[方块名称]_rotating.png`
- 发光效果: `[方块名称]_glow.png`

### 连接贴图
对于需要与其他方块连接的组件（如机械轴），可以添加连接贴图：
- 连接端点: `[方块名称]_end.png`
- 连接中部: `[方块名称]_middle.png`

## 代码集成
确保您的方块代码中正确引用了贴图：
```java
// 在方块定义中，Mindustry会自动加载同名贴图
BlockName = new Block("block_name") {{
    // 方块属性...
}};
```

## 测试贴图
1. 将PNG文件放入正确的目录
2. 重新编译并运行游戏
3. 在游戏中测试方块外观

## 常见问题

### 贴图不显示
- 检查文件名是否与方块名称完全一致
- 确保文件位于正确的目录结构中
- 验证文件格式是否为PNG

### 贴图尺寸错误
- 确保贴图尺寸与方块大小匹配
- 1x1方块使用32x32像素，2x2方块使用64x64像素

### 贴图风格不匹配
- 参考Mindustry原版方块的风格和色彩
- 保持一致的像素密度和线条粗细
